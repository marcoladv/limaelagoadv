
# Como adicionar fotos dos advogados

1. Prepare uma imagem quadrada para cada advogado (ex: 300x300px).
2. Nomeie como `marco.jpg` e `esther.jpg`.
3. Suba os arquivos no repositório (GitHub).
4. No `index.html`, substitua os blocos <div class="foto-circular">Foto</div> por:

   <img src="marco.jpg" class="foto-circular" alt="Foto Marco">
   <img src="esther.jpg" class="foto-circular" alt="Foto Esther">

5. As imagens serão exibidas como círculos automaticamente.
